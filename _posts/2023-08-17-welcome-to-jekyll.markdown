```java
"변수란, 단 하나의 값을 저장할 수 있는 메모리 공간"
```

`변수의 선언과 초기화`

변수를 사용하려면 먼저 변수를 선언해야 하는데, 변수의 선언방법은 다음과 같다.
```java
int age; // int 타입의 age라는 변수 선언

or

int age = 0; // int 타입의 age라는 변수 초기화
```

여기서 'int' 는 변수타입 'age' 는 변수이름 or 변수명 이다.
'변수타입은' 변수에 저장될 값이 어떤 '타입(type)'인지를 지정하는 것이다.
'변수이름'은 말 그대로 변수에 이름을 붙인 것이다.
변수는 '값을 저장할 수 있는 메모리 공간'이므로 변수의 이름은 메모리 공간에 이름을 붙여주는 것이다. 그래야 그 이름을 이용해서 저장공간에 값을 저장하고,
저장된 값을 읽어오기도 할 수 있는 것이다.

변수를 선언한 이후부터는 변수를 사용 가능하나 그 전에 반드시 '초기화'를 해야 한다.
메모리는 여러 프로그램이 공유하는 자원이므로 전에 다른 프로그램에 의해 '알 수없는 값'이 남아있을 수 있기 때문이다.
